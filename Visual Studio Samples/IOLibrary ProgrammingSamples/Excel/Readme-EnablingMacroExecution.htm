Enabling Macro Execution

When loading any of the Excel samples, you may see a message box that says,

Macros are disabled because the security level is set to High and a digitally signed Trusted Certificate is not attached to the macros. To run the macros, change the security level to a lower setting (not recommended), or request the macros be signed by the author using a certificate issued by a Certificate Authority.

You must enable macro execution in Excel for samples to run.

In Office 2002, you must check the checkbox in Excel:

Tools -> Macro -> Security -> Trusted Sources -> "Trust access to Visual Basic Project".

In Office 2003, you must check the checkbox in Excel:

Tools -> Macro -> Security -> Trusted Publishers -> "Trust access to Visual Basic Project".
