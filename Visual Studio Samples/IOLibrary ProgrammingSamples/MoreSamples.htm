Finding Programming Samples

This Read Me file describes several sources of programming samples that can help you get started programming your instruments. These sources include:

  • Agilent IO Libraries Suite (the folder in which you found this file)

  • The Agilent Developer Network Web site

  • The CD-ROM that came with your instrument

  • Other Agilent software products

  • Instrument drivers

Note: If you have difficulty finding sample code for your instrument model, consider adapting programming samples for a closely-related model in the same instrument family. For instance, most sample programs for the Agilent 33220A function generator can also be used with the Agilent 33250A.

Agilent IO Libraries Suite Programming Samples

Agilent IO Libraries Suite includes a large set of samples that demonstrate instrument programming techniques. These samples are installed in your IO Libraries Suite installation directory (default: C:\Program Files\Agilent\IO Libraries Suite) in the ProgrammingSamples folder; within this folder, they are arranged by programming language or environment, then by the I/O API used in the sample code. Represented languages and environments include C, C#, Visual Basic (version 6), Visual Basic .NET, and Excel. The samples illustrate the use of VISA, VISA COM, and SICL for instrument programming.

Example: To find programming examples in Visual Basic (version 6) utilizing the VISA COM library, navigate to VB6\VISACOM.  Within that folder, there are several subfolders containing programming samples for various instrument models and instrument families.

Agilent Developer Network

A wide variety of test and measurement programming samples are available on the Agilent Developer Network (ADN) Web site, at http://www.agilent.com/find/ADNCodeSamples. These include demonstrations of programming techniques, use of specific instrument models and families, USB instrument programming, and more. The ADN site also includes downloadable instrument drivers, technical white papers, and more.

Your Instrument CD

If you have purchased an instrument from Agilent or from another vendor, you probably received a CD-ROM with instrument-specific software and documentation. The contents of that CD may include programming samples for your instrument as well as instrument drivers, manuals, and application software. Browse the instrument CD-ROM to find sample code that fits your needs.

Other Agilent Software Products

Agilent VEE Pro includes many programming samples, installed under the VEE installation folder in the subfolder examples. You can access these samples easily from within VEE Pro by clicking Help > Open Example ...

Agilent T&M Toolkit also includes a large number of samples, installed under the T&M Toolkit installation folder in the subfolder Samples. To see a list of these programming samples from within Microsoft Visual Studio .NET, click T&M Toolkit > Getting Started and then click the Samples tab.

Evaluation versions of VEE Pro and T&M Toolkit are available on the Automation-Ready CD with Agilent IO Libraries Suite. You can also download evaluation versions and find purchasing information at www.agilent.com/find/vee and www.agilent.com/find/toolkit respectively.

Instrument Drivers

Many instrument drivers include programming samples that demonstrate the use of those drivers to perform general and instrument-specific functions. There are many instrument drivers available on the Agilent Developer Network, and there are several provided on the Automation-Ready CD with Agilent IO Libraries Suite (in the Sample Instrument Drivers folder).

Programming Samples Installed with IVI-COM Instrument Drivers

If you have installed Agilent IVI-COM drivers for your instruments, sample code was also installed with these drivers. (IVI-COM drivers from other vendors may also include sample code.) The standard location for IVI-COM programming samples is <IVI Component Path>\Drivers\<Instrument Model>\Examples

You can determine the IVI Component Path by clicking the IO Control and clicking Documentation > Installation and Path Information, or by selecting Help > Installation Information in Connection Expert.

Example: If you have installed an IVI-COM instrument driver for the Agilent 33220A arbitrary waveform generator, and if you used the default IVI Components folder, you can find instrument-specific programming samples in C:\Program Files\IVI\Drivers\Agilent33220\Examples. You can also access these samples by clicking Start > Programs > IVI > Agilent33220 > Agilent33220 Examples.

Programming Samples Installed with VXIplug&play Instrument Drivers

Many VXIplug&play instrument drivers include programming samples. Their installation location is not defined by standards, but the default installation location for VXIplug&play drivers is under the VISA Path, which you can determine by clicking the IO Control and clicking Documentation > Installation and Path Information, or by selecting Help > Installation Information in Connection Expert.

Example: If you have installed a VXIplug&play instrument driver for the Agilent 33250A arbitrary waveform generator, and if you used the default VISA folder on a Windows XP PC, you can find instrument-specific programming samples in C:\Program Files\VISA\winnt\AG33250A\examples. Additional sample code is available in the instrument driver's Help file, which you can access by clicking Start > Programs > VXIPNP > Agilent33250A & 33220A Help.
